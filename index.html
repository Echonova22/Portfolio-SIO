Mon alternance

Le Département d'Ille-et-Vilaine est composé de 27 321 agents. Mes missions : gestion des tickets, installations et support utilisateurs.

Projet 1 : Installation d'un serveur Active Directory

Objectif : Mettre en place une infrastructure serveur complète sous Windows Server avec les services suivants :

  • Active Directory (AD) : création du domaine, gestion des utilisateurs, des groupes et des unités d'organisation.
  • DHCP : configuration d'un serveur DHCP pour l'attribution automatique des adresses IP aux postes clients du réseau.
  • DNS : mise en place du service de résolution de noms pour le bon fonctionnement du domaine Active Directory.
  • Serveur de partage de fichiers : création de dossiers partagés avec gestion des droits d'accès par groupes d'utilisateurs.
  • GPO (Group Policy Objects) : déploiement de stratégies de groupe pour uniformiser les configurations des postes clients (fond d'écran, restrictions, scripts de connexion, etc.).

Technologies utilisées : Windows Server 2019, Active Directory Domain Services (AD DS), DHCP Server, DNS Server, Group Policy Management Console (GPMC).

Résultat : Infrastructure fonctionnelle permettant la gestion centralisée des utilisateurs, des ressources réseau et des politiques de sécurité.

Projet 2 : Mise en place d'un serveur GLPI

Objectif : Déployer et configurer un serveur GLPI (Gestionnaire Libre de Parc Informatique) pour la gestion du parc informatique.

  • Installation du serveur GLPI : déploiement sur un serveur Linux avec configuration de la pile LAMP (Linux, Apache, MySQL, PHP).
  • Configuration de GLPI : paramétrage des catégories de tickets, des entités, des profils utilisateurs et des notifications par mail.
  • Ajout des utilisateurs : création des comptes utilisateurs, affectation des rôles et des droits d'accès selon les profils (technicien, administrateur, utilisateur).
  • Inventaire des postes : ajout et recensement des équipements informatiques (ordinateurs, périphériques, logiciels) dans la base de données GLPI pour assurer un suivi complet du parc.

Technologies utilisées : Debian Linux, Apache2, MySQL/MariaDB, PHP, GLPI, FusionInventory (agent d'inventaire automatique).

Résultat : Outil de gestion de parc opérationnel permettant de centraliser les demandes d'assistance, de suivre les incidents et d'inventorier l'ensemble des actifs informatiques de l'organisation.
